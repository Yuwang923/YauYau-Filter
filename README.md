## Yauyau filter 简介
Yau-Yau滤波器是一种处理非线性滤波问题的数学方法，主要用于解决非线性动态系统中噪声数据的滤波问题，旨在实时、无记忆地求解Duncan–Mortensen–Zakai（DMZ）方程。该滤波器通过将滤波问题转化为Kolmogorov方程的求解，来实现对系统状态的实时估计。其应用场景广泛，包括军事雷达系统的目标跟踪、自动驾驶车辆的状态估计、以及金融数据的动态分析等场合，为实时信号处理和状态预测提供了精确的非线性滤波手段。
