## Yauyau filter 简介
非线性滤波问题在军事、工程和商业领域有着重要的应用。非线性滤波的核心问题是在实时和非记忆性方式下求解邓肯-莫滕森-扎凯（DMZ）方程。Yau-Yau滤波器是Yau-Yau方法求解高维非线性滤波问题的有效数值算法。
